# 🚀 Agency-AI

A modern, elegant, and fully responsive AI-powered agency website built with **React.js**, **Tailwind CSS**, and **Framer Motion**. Designed with performance, smooth animations, and a seamless dark mode experience in mind.

---

## 🌐 Live Demo

👉 [https://agency-ai-using-react-js-tailwind-c.vercel.app/)

---

## ✨ Features

* 🎨 Modern & elegant UI design
* 📱 Fully responsive (mobile, tablet, desktop)
* 🌙 Dark mode support (toggle enabled)
* ⚡ Fast performance (optimized build)
* 🎞 Smooth animations using Framer Motion
* 🧩 Component-based architecture
* 🧠 AI-themed agency layout

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Deployment:** Vercel

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/jossy21-coder/Agency_AI-using-React.js-TailwindCSS-Framer-Motion.git
```

2. Navigate to the project directory

```bash
cd agency-ai
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```

---

## 🚀 Build & Deployment

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

### Deploy on Vercel

1. Push your project to GitHub
2. Go to [https://vercel.com](https://vercel.com)
3. Import your repository
4. Click **Deploy**

---

## 🌙 Dark Mode Implementation

Dark mode is implemented using Tailwind's `dark` class strategy.

Example:

```js
document.documentElement.classList.toggle("dark");
```

---

## 🎞 Animations

Framer Motion is used for smooth UI interactions and transitions.

Example:

```jsx
import { motion } from "framer-motion";

<motion.div
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.5 }}
>
  Content
</motion.div>
```
